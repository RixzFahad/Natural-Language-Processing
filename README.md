```
Raw Text → Sentence Tokenization → Word Tokenization
→ Remove Punctuation → Remove Stop Words → Stemming / Lemmatization
```

---

## 🧩 Key Concepts

<details>
<summary><b>Stemming vs Lemmatization</b></summary>

| | Stemming | Lemmatization |
|---|---|---|
| Output | May not be real word | Always real word |
| Example | `studies → studi` | `studies → study` |
| Failure | `ran → ran` ❌ | `ran → run` ✅ |

</details>

<details>
<summary><b>Representations: BoW → BERT</b></summary>

| Method | Contextual? |
|---|---|
| Bag of Words / TF-IDF | ❌ |
| Word2Vec / GloVe | ❌ Static |
| BERT | ✅ Dynamic |

</details>
